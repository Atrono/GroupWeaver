Key UX improvements — Graph explorer
Encoded node language. Shape = object class (circle group / square OU / glyph user), colored ring = scope (GG·DL·UG), corner badge = audit state. Type is readable before you read a label.
Focus + context. Root scales up and gets the selection halo; unexpanded membership stays a dashed "+14" frontier so the canvas stays calm until you ask for more.
Legible at scale. Ctrl+K search, type & "issues-only" filters, persistent legend and minimap keep large directories navigable.
Trust & privacy. Read-only lock in the chrome; the detail panel shows only the whitelisted attribute set with an explicit "hidden by privacy baseline" note.

WHAT'S WRONG
svc_backup is a direct member of DL_Finance_RW. Under AGDLP, accounts live in global groups (G); domain-local groups (DL) should contain only groups — not users.
WHY IT MATTERS
Direct user-in-DL membership scatters access control, makes audits harder, and breaks the A→G→DL→P chain the rest of your directory follows.
HOW TO FIX
1Add svc_backup to a global group, e.g. GG_Finance_Svc.
2Make that global group a member of DL_Finance_RW.
3Remove the direct user membership.

Key UX improvements — Audit / findings
Health first. A single score plus critical / warning / passing tiles answer "is my directory clean?" before any drilling.
Table or graph. A view toggle flips the same findings between a dense, sortable, multi-select table and the node graph — the right tool for the directory's size.
Triage at scale. Select rows and Acknowledge / Suppress in bulk; a suppression becomes an auditable, reversible exception in the ruleset's ignore list, so known-good structures stop nagging.
Plain-language + copyable fix. What's wrong, why it matters, and a numbered fix — including a read-only PowerShell snippet to run in your own shell. GroupWeaver never changes AD.

Key UX improvements — Light variant
Same IA, retuned contrast. Identical layout and node language to the dark theme — only surfaces, lines and accent are re-derived for white backgrounds.
Accessible in bright rooms. Deeper traffic-light and scope hues hold their meaning on light surfaces (and on a projector during reviews).
Theme is one switch. Built on theme tokens, so dark ⇄ light is a single toggle — no divergent screens to maintain.
